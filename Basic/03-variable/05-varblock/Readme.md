Orignal:
```
var (
		age       int     = 10
		weight    float32 = 60.5
		name      string  = "Gunjan"
		isStudent bool    = true
	)
```    
Better and short 

var (
	age       = 10
	weight    = 60.5
	name      = "Gunjan"
	isStudent = true
)

use short declaration (:=) inside main(): with whole declarion inside main func.

age := 10
weight := 60.5
name := "Gunjan"
isStudent := true

// Note: You can also use var keyword in the block without parentheses but it is not a good practice.
